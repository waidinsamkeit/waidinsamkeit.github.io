1.1 索引切片

主要是说索引切片这个问题

主要说明索引的切片
print(lists[0],type(lists[0])) # index0的存储数据类型
print(lists[1],type(lists[1])) # index1的存储数据类型
print([lists[:4]]) # 取值0-4索引内容
print(lists[::2])  # 步长为2 隔1取一
print(lists[-1:2:-2])

    lists.append(add) # 默认追加最后
print(lists)
lists.insert(1,'population') # 在索引下标1前插入
lists.extend('auther') # 可迭代追加，只拆分最小元素

1.3 列表的删除操作

# // 删除
lists.pop(0) # 删除索引为0的数据,可以返回删除值
lists.clear()  # 清空列表内容 
del lists  # 可以切片 可以索引删除,包括内存级别删除索引
del lists[0] # 删除索引0的数据
del lists[:2] # 删除0-1的数据
del lists[1::2]
