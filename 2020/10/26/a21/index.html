发布于：2020年10月26日

Zabbix企业级分布式监控部署

  • 更新与2020-10-20-10.53

本次采用Zabbix5.0LTS进行

  • 新数据，新架构，新服务，新监控

  • 网络监控，服务器监控，云监控，应用监控，服务监控

  • 监控100,000+主机

  • 采集100,000,000+指标

  • 分布式监控，防火墙后监控，高可用，无单点故障

行了，不吹他了，直接部署了

准备环境

Zabbix-server 192.168.1.100
Zabbix-agent 192.168.1.101
  • 内存1Gb以上
  • 以下命令两台主机全部执行
echo "192.168.1.100 Zabbix-server" >> /etc/hosts
echo "192.168.1.101 Zabbix-agent" >> /etc/hosts
wget -o /etc/yum.repos.d/Centos-SCLo-rh.repo https://gitee.com/brave_heart_one/rh-scl/blob/master/Centos-SCLo-rh.repo
关闭selinux和firewalld
reboot

注意PHP的版本

The minimum required PHP version has been upped to 7.2.0 from 5.4.0.

Zabbix5.x支持的PHP版本已经不支持5.4.0 需要支持7.2

需要有LAMP环境，本人搭建使用一下环境

  • mysql8.0
  • php-7.2
  • nginx1.16

1.1 Zabbix-server安装服务

简要的来说zabbix的服务分为server和agent
zabbix server可以通过SNMP，zabbix agent，ping，端口监视等方法提供对远程服务器/网络状态的监视，数据收集等功能，它可以运行在Linux，Solaris，HP-UX，AIX，Free BSD，Open BSD，OS X等平台上。

以下安装方法仅仅适用于Centos与Redhat发行版本

[root@Zabbix-server ~]# rpm -Uvh https://repo.zabbix.com/zabbix/5.0/rhel/7/x86_64/zabbix-release-5.0-1.el7.noarch.rpm
[root@Zabbix-server ~]# yum clean all
[root@Zabbix-server ~]# yum install zabbix-server-mysql
[root@Zabbix-server ~]# yum install -y centos-release-scl
1.1.1配置zabbix仓库
[root@Zabbix-agent ~]# vim /etc/yum.repos.d/zabbix.repo
enabled=1  //将里面的所有enable改为1
1.1.2开始安装zabbix-web
[root@Zabbix-server ~]# yum install zabbix-web-mysql-scl zabbix-nginx-conf-scl -y
[root@Zabbix-server ~]# scl enable rh-php72 bash //运行php,此步骤仅限scl安装的php启动
1.1.3 安装mysql8.0
[root@Zabbix-server ~]# wget https://dev.mysql.com/get/mysql80-community-release-el7-3.noarch.rpm
[root@x-server ~]# rpm -ivh mysql80-community-release-el7-3.noarch.rpm
[root@Zabbix-server ~]# systemctl enable mysqld
[root@Zabbix-server ~]# systemctl restart mysqld
[root@Zabbix-server ~]# mysql -u root  //切记修改密码

怎么修改mysql密码我就不多说了

传送门

1.4导入zabbix数据库
[root@Zabbix-server ~]# mysql -u root -p
mysql> create database zabbix character set utf8 collate utf8_bin;
mysql> create user 'zabbix'@'localhost' identified with mysql_native_password by '123.comA'; //zabbix库允许用户zabbix的登陆密码123.comA
mysql> grant all privileges on zabbix.* to zabbix@localhost;
mysql> quit;
[root@Zabbix-server ~]# zcat /usr/share/doc/zabbix-server-mysql*/create.sql.gz | mysql -uzabbix -p zabbix
[root@Zabbix-server ~]# vim /etc/zabbix/zabbix_server.conf
DBPassword=123.comA // zabbix用户的密码
1.5为zabbix配置php
[root@Zabbix-server ~]# vim /etc/opt/rh/rh-nginx116/nginx/conf.d/zabbix.conf
        listen          8088; //监听端口为8088
#        server_name     example.com; //此项可以不修改
[root@Zabbix-server ~]# vim /etc/opt/rh/rh-php72/php-fpm.d/zabbix.conf
listen.acl_users = apache,nginx // 在apache用户后面加入nginx
php_value[date.timezone] = Asia/shanghai //去掉;并且加入自己的时区
1.6 启动zabbix-server
[root@Zabbix-server ~]# systemctl enable zabbix-server  rh-nginx116-nginx rh-php72-php-fpm
[root@Zabbix-server ~]# systemctl restart zabbix-server  rh-nginx116-nginx rh-php72-php-fpm
1.7 访问zabbix
访问 http://192.168.20.22:8088

自己按照提示配置就行
