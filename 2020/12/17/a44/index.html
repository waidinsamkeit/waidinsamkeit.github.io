SSH禁止root远程登录+提权

Rainy City
Linux

发布于：2020年12月17日

安全防护策略

  • 禁止root登录系统
  • 建立独立账户,使用单独权限

1.0 禁止root登录

修改sshd配置文件

vim /etc/ssh/sshd_config
找到 PermitRootLogin
#PermitRootLogin yes 修改为 PermitRootLogin no

注意: 这个时候不要重启你的ssh服务,否则你可能只能通过VNC云客户端登录

2.0 赋予root权限

vim  /etc/sudoers
# Allows people in group wheel to run all commands
%wheel    ALL=(ALL)    ALL # 取消注释
useradd test # 新增一个测试用户

usermod -g root  test # 把test用户放到root组
