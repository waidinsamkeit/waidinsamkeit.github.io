CentOS镜像使用帮助

  • CentOS
  • CentOS 8（非 Stream 版本）已被官方移除出该仓库。如有需要，请使用 CentOS-Vault 镜像。
  • CentOS 9 Stream 及以后的版本的镜像位于 CentOS-Stream 。

对于 CentOS 8 Stream，使用以下命令替换默认的配置

1sudo sed -e 's|^mirrorlist=|#mirrorlist=|g' \
2         -e 's|^#baseurl=http://mirror.centos.org/$contentdir|baseurl=https://mirrors.ustc.edu.cn/centos|g' \
3         -i.bak \
4         /etc/yum.repos.d/CentOS-Stream-AppStream.repo \
5         /etc/yum.repos.d/CentOS-Stream-BaseOS.repo \
6         /etc/yum.repos.d/CentOS-Stream-Extras.repo \
7         /etc/yum.repos.d/CentOS-Stream-PowerTools.repo

对于 CentOS 7，使用以下命令替换默认配置

1sudo sed -e 's|^mirrorlist=|#mirrorlist=|g' \
2         -e 's|^#baseurl=http://mirror.centos.org/centos|baseurl=https://mirrors.ustc.edu.cn/centos|g' \
3         -i.bak \
4         /etc/yum.repos.d/CentOS-Base.repo
